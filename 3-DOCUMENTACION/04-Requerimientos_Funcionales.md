# Requerimientos Funcionales

## RF01
El sistema deberá permitir registrar miembros.

## RF02
El sistema deberá permitir iniciar sesión.

## RF03
El sistema deberá permitir gestionar talleres.

## RF04
El sistema deberá permitir gestionar eventos.

## RF05
El sistema deberá permitir gestionar proyectos.

## RF06
El sistema deberá permitir consultar información de talleres.

## RF07
El sistema deberá permitir consultar eventos programados.
