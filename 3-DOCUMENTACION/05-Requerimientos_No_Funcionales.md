# Requerimientos No Funcionales

## RNF01
El sistema deberá estar disponible las 24 horas.

## RNF02
La interfaz deberá ser intuitiva y fácil de usar.

## RNF03
Las credenciales deberán almacenarse de forma segura.

## RNF04
El tiempo de respuesta deberá ser menor a 3 segundos.

## RNF05
El sistema deberá ser compatible con navegadores modernos.
