# Diagramas de Comportamiento

## Descripción

Esta carpeta contiene los diagramas que representan el comportamiento dinámico del sistema.

## Diagramas incluidos

- Casos de Uso
- Diagramas de Secuencia
- Especificaciones de Casos de Uso

## Objetivo

Describir las funcionalidades del sistema y las interacciones entre actores y componentes.
