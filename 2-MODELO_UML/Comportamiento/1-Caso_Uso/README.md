# Casos de Uso

## Descripción

Aquí se encuentran los diagramas de casos de uso del Sistema de Gestión del Club de Programación.

## Casos implementados

- CU01 Registrar miembro
- CU02 Iniciar sesión
- CU03 Gestionar talleres
- CU04 Gestionar eventos
- CU05 Gestionar proyectos
- CU06 Gestionar usuarios
- CU07 Inscribirse a taller

## Objetivo

Representar gráficamente las funcionalidades disponibles para los usuarios del sistema.
