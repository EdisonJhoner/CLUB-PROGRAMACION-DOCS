# Sistema de Gestión para un Club de Programación

## Descripción

El Sistema de Gestión para un Club de Programación permite administrar miembros, talleres, eventos y proyectos desarrollados dentro de la organización.

## Objetivo

Facilitar la gestión académica y administrativa del club mediante herramientas digitales.

## Funcionalidades

* Registro de miembros.
* Inicio de sesión.
* Gestión de talleres.
* Gestión de eventos.
* Gestión de proyectos.

## Tecnologías

* IntelliJ IDEA
* Java
* Maven
* PlantUML
